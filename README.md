# Titanic Survival Prediction

This project analyzes the Titanic dataset to predict passenger survival using machine learning. It includes data cleaning, feature engineering, and model selection with scikit-learn.

## Features

- Data preprocessing (handling missing values, encoding categorical variables)
- Feature engineering (e.g., Title extraction, FamilySize, IsAlone, FarePerPerson)
- Model training and evaluation (Random Forest, Gradient Boosting, Logistic Regression, Ensemble Voting)
- Hyperparameter tuning with GridSearchCV
- Submission file generation for Kaggle

## Usage

1. Clone the repository.
2. Place the Titanic dataset in the `data/` folder.
3. Run the Jupyter notebook in `notebooks/` for step-by-step analysis.
4. Submission files are saved as `submission.csv` and `submission_v2.csv`.

## Requirements

- Python 3.8+
- pandas, scikit-learn, matplotlib, seaborn
